# Primeiro-Reposit-rio
Repositório criado para documentar práticas e exercícios realizados durante meu processo de aprendizado em Cyber Segurança. 
Vou documentar ferramentas, comandos e práticas que realizo durante a aprendizagem no Mulher Digital. 
Aqui estão os insights que eu extrair do curso introdutório de cybersegurança da Cisco:
Ameaças e Malware
Vírus: Malware que precisa se anexar a arquivos e depende da ação do usuário.
Worm: Replica-se sozinho pela rede explorando vulnerabilidades.
Trojan: Disfarça-se de software legítimo para enganar o usuário.
Spyware: Espiona atividades e coleta dados do usuário.
Botnet: Rede de dispositivos infectados controlados remotamente.
Backdoor: Porta secreta para acesso futuro ao sistema.
APT: Ataque avançado, furtivo e de longo prazo contra alvos específicos.
Ataques
DoS/DDoS: Sobrecarrega serviços para torná-los indisponíveis.
Phishing: Engana usuários para roubar informações.
Engenharia Social: Manipulação psicológica para obter dados.
SEO Poisoning: Manipula buscas para direcionar vítimas a sites maliciosos.
Proteção
Firewall: Filtra tráfego de rede com base em regras.
Antivírus: Detecta e remove malwares.
IDS: Detecta ataques e gera alertas.
IPS: Detecta e bloqueia ataques automaticamente.
VPN: Cria túnel criptografado para navegação segura.
DLP: Impede vazamento de dados sensíveis.
SIEM: Centraliza e analisa logs de segurança.
